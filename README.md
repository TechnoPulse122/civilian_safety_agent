# civilian_safety_agent
An AI-powered multi-agent system designed to help civilians stay safe during emergencies such as war, natural disasters, and accidents by providing real-time, actionable guidance.
Overview
    In critical situations, people often panic and lack clear guidance.
    This project solves that problem by using a multi-agent AI system that:
    
    Understands the user’s situation
    Researches safety protocols
    Generates step-by-step actions
    Delivers clear and calm instructions
Features
  🤖 Multi-Agent AI Architecture
  🧠 Intelligent Situation Understanding
  🌐 Real-Time Knowledge Retrieval (Wikipedia)
  📋 Automatic Task Checklist Generation
  ⚡ Priority-Based Action Guidance
  💬 Clear & Structured Emergency Responses
  🔄 End-to-End Workflow Automation
  🌍 Supports Multiple Emergency Scenarios
  🌐 API-Ready Deployment
  🏗️ Architecture

The system uses a Sequential Multi-Agent Workflow:

  Root Agent (Safety Greeter):
    Interacts with user
    Captures emergency input
  Safety Researcher Agent:
    Gathers safety information using tools
  Task Coordinator Agent:
    Converts data into actionable checklist
  Emergency Formatter Agent:
    Formats final response clearly
  Workflow

User Input → Research Agent → Task Agent → Formatter Agent → Final Output
